# 📊 E-commerce Sales Dashboard (Excel)

## 🔹 Overview

This project focuses on analyzing e-commerce sales data using Microsoft Excel to uncover key business insights and support data-driven decision-making. The dashboard provides an interactive view of sales performance across different categories, regions, and customer segments.

---

## 🔹 Objectives

* Analyze sales trends and performance metrics
* Identify top-performing products and regions
* Track revenue, profit, and order patterns
* Build an interactive dashboard for easy data exploration

---

## 🔹 Tools & Techniques Used

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Data Cleaning & Validation
* Conditional Formatting
* Dashboard Design
* Basic Data Analysis

---

## 🔹 Key Features

* 📌 Interactive dashboard with slicers and filters
* 📈 Visual representation of KPIs (Revenue, Profit, Orders)
* 🌍 Region-wise and category-wise performance analysis
* 🔍 Easy drill-down for detailed insights

---

## 🔹 Key Insights

* Top product categories contributed significantly to overall revenue
* Identified fluctuations in monthly sales trends
* Highlighted high-performing regions driving business growth
* Detected areas with lower profitability for improvement
